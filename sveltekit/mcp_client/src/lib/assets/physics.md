# 4-DoF Robot Arm：物理制御および動作規定仕様書

本ドキュメントは、4自由度（4-DoF）ロボットアームにおける物理的な動作制御、および安全性確保のための論理フレームワークを規定するものです。運用にあたっては、別途提供される**物体カタログ**、および**ワークスペース画像取得ツール**（物体検出ツール YOLO 内蔵）、**ロボット操作ツール**を活用してください。

---

## 1. 干渉回避と経路計画 (Collision Avoidance)
ロボットアームのエンドエフェクターおよび把持物体が、周囲の環境や他の物体と物理的接触を起こさないための計算モデルです。

* **ワークスペース画像取得ツールによる物体認識:** 本ツールに内蔵された **YOLO** により、各物体は「円筒」として近似され、以下のデータがリアルタイムで出力されます。これらを基に経路上の障害物判定を行います。
    * **物体半径 ($r_{obj}$):** 円筒とみなした際の最大半径。
    * **物体高さ ($h_{obj}$):** 接地面からの垂直距離。
    * **物体接地面中央位置座標 ($P_{base}$):** 物体が設置されている中心の $x, y, z$ 座標。
* **近接優先の原則:** ワークスペース内に複数の物体が検出された場合、ロボットのベース座標（原点）に最も近い物体から優先的に処理し、アクセス時の障害物リスクを最小化します。
* **安全高度 (Safe Altitude) の動的決定:** 移動時のZ軸座標は、検出された物体の高さを基に動的に決定します。ピック座標を $P_{start}$、プレイス座標を $P_{end}$ としたとき、安全高度 $z_{safe}$ は以下の数式に基づき算出します。
  $$z_{safe} = \max(z_{start}, z_{end}, z_{obstacles}) + 20\text{mm}$$
* **幾何学的障害物判定:** 移動経路（$P_{start}$ と $P_{end}$ を結ぶ直線）上に物体 $O_i$ が存在するかは、画像取得ツールから得られた $P_{base}$ と $r_{obj}$ に安全マージンを加えた円を用いて判定します。この円と経路が交差する場合、その物体の高さ $h_{obj}$ を安全高度の計算に算入します。

## 2. 物体特性に応じた把握制御 (Grasp Control)
**ワークスペース画像取得ツール**から得られたリアルタイムの寸法データと、**物体カタログ**の物性データを組み合わせて制御します。

* **把持基準幅 ($W_{object}$) の算定:**
  画像取得ツールにて物体を円筒とみなして取得した半径 $r_{obj}$ を用い、物体の自然長（幅）を以下のように定義します。
  $$W_{object} = r_{obj} \times 2$$
* **弾性体 (Elastic Objects) への対応:**
  **物体カタログ**上で弾性体と指定されている物体（メラニンスポンジ等）を扱う場合、上記で算出した $W_{object}$ に対し、グリッパー幅 $W_{grip}$ を以下の比率で制御します。
  $$W_{grip} \approx W_{object} \times 0.8$$
  この「20%の圧縮」を保持した状態で、**ロボット操作ツール**を介した移動操作を行います。

## 3. 座標系の定義と接地シーケンス
高精度なハンドリングを実現するため、座標計算と最終的な接地動作を規定します。

* **TCP (Tool Center Point) 座標の確定:**
  ピックおよびプレイス時の目標座標は、画像取得ツールから得られた**物体接地面中央位置座標 ($P_{base}$)** および検出された**物体高さ ($h_{obj}$)** を直接参照し、**ロボット操作ツール**上で最終的な目標TCP座標として確定させます。
* **段階的接地マージン (Soft Landing):**
  プレイス動作において、作業台との不要な衝撃を避けるため、物体の底面が接地する直前（目標 $z$ 座標 +10mm）で、**ロボット操作ツール**を通じて「微速下降モード」へ切り替えるシーケンスを標準動作として定義します。
